# 打包资源说明

此目录存放打包所需的资源文件：

## 图标文件要求

### macOS
- `icon.icns`：应用图标，尺寸 512x512px
- `tray-icon.icns`：托盘图标，尺寸 16x16px / 32x32px（2x）

### Windows
- `icon.ico`：应用图标，包含 256x256、128x128、64x64、48x48、32x32、16x16 尺寸
- `tray-icon.ico`：托盘图标，尺寸 16x16px

### 通用
- `icon.png`：通用 PNG 图标，尺寸 512x512px
- `tray-icon.png`：通用托盘图标，尺寸 32x32px

## 其他资源
- `background.png`：dmg 安装包背景图（macOS）
- `license.txt`：软件许可协议

## 生成图标建议
使用在线工具生成多尺寸图标：
