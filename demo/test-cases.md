# 演示测试用例集

## 接口调用测试用例

### TC-API-001: 导出订单报表
- **优先级**：高
- **前置条件**：Mock服务器已启动，API映射已配置
- **测试步骤**：
  1. 选择执行模式为"智能模式"
  2. 输入指令："导出2026年3月1日到2026年3月31日的订单报表，格式为excel"
  3. 等待执行完成
- **预期结果**：
  - 执行模式显示为"api"
  - 接口调用成功，返回报表信息
  - 包含文件名、大小、订单总数、总金额等字段
  - 执行时间≤3秒

### TC-API-002: 查询用户信息
- **优先级**：高
- **测试步骤**：
  1. 输入指令："查询用户ID是1的详细信息"
  2. 查看执行结果
- **预期结果**：
  - 成功返回用户信息
  - 包含姓名、邮箱、电话、注册时间等字段
  - 姓名为"张三"

### TC-API-003: 创建订单
- **优先级**：高
- **测试步骤**：
  1. 输入指令："创建一个新订单，产品ID是P001，数量2个，金额1999元，客户ID是C001"
  2. 查看执行结果
- **预期结果**：
  - 订单创建成功
  - 返回订单ID，状态为"created"
  - 订单信息与输入参数一致

### TC-API-004: 接口参数缺失
- **优先级**：中
- **测试步骤**：
  1. 输入指令："导出订单报表"（不提供日期参数）
  2. 查看执行结果
- **预期结果**：
  - Agent识别参数缺失
  - 返回询问："请问需要导出哪个时间段的订单报表？请提供开始日期和结束日期。"

## MCP服务测试用例

### TC-MCP-001: 财务分析
- **优先级**：高
- **前置条件**：MCP服务已配置
- **测试步骤**：
  1. 输入指令："做2026年第一季度的财务分析"
  2. 查看执行结果
- **预期结果**：
  - 执行模式显示为"mcp"
  - 调用财务分析能力成功
  - 返回营收、成本、利润、利润率等核心指标
  - 包含至少3条优化建议
  - 执行时间≤3秒

### TC-MCP-002: 预算计算
- **优先级**：中
- **测试步骤**：
  1. 输入指令："计算市场部2026年第一季度的预算使用情况"
  2. 查看执行结果
- **预期结果**：
  - 返回总预算、已分配、剩余金额
  - 包含各预算项目的使用明细
  - 数据合理

### TC-MCP-003: 税费计算
- **优先级**：高
- **测试步骤**：
  1. 输入指令："计算3月份的税费，营收80万，成本30万，可抵扣20万"
  2. 查看执行结果
- **预期结果**：
  - 正确计算应纳税额：(80-30-20) × 25% = 7.5万
  - 返回应缴税额、税率、申报截止日期
  - 包含节税建议

### TC-MCP-004: 能力不存在
- **优先级**：低

  1. 输入指令："做一个人力资源分析"
  2. 查看执行结果
- **预期结果**：
  - Agent识别没有对应MCP能力
  - 提示："暂不支持人力资源分析能力，是否需要使用网页自动化方式操作？"

## 模式切换测试用例

### TC-MODE-001: 强制API模式
- **优先级**：中
- **测试步骤**：
  1. 切换执行模式为"接口优先"
  2. 输入有对应接口的指令："导出2026年2月的订单报表"
  3. 查看执行模式
- **预期结果**：
  - 强制使用API模式执行
  - 即使接口失败也不降级为网页自动化

### TC-MODE-002: 强制网页模式
- **优先级**：中
- **测试步骤**：
  1. 切换执行模式为"网页操作"
  2. 输入有对应接口的指令："查询用户ID为2的信息"
  3. 查看执行模式
- **预期结果**：
  - 强制使用网页自动化模式执行
  - 生成网页操作步骤，不调用接口

### TC-MODE-003: 自动模式降级
- **优先级**：高
- **测试步骤**：
  1. 关闭Mock服务器（模拟接口不可用）
  2. 切换执行模式为"智能模式"
  3. 输入有对应接口的指令："导出订单报表"
  4. 查看执行过程
- **预期结果**：
  - 首先尝试API调用，返回失败
  - 自动降级为网页自动化模式
  - 提示："接口调用失败，将使用网页自动化方式执行"

## 异常场景测试用例

### TC-ERROR-001: 网络异常
- **优先级**：高
- **测试步骤**：
  1. 断开网络连接
  2. 输入需要调用接口的指令
  3. 查看错误提示
- **预期结果**：
  - 友好提示："网络请求失败，请检查网络连接"
  - 应用不崩溃，可继续使用

### TC-ERROR-002: 鉴权失败
- **优先级**：中
- **测试步骤**：
  1. 在设置中清空认证Token
  2. 输入需要鉴权的接口指令
  3. 查看错误提示
- **预期结果**：
  - 提示："身份验证失败，请检查登录状态或重新配置Token"

### TC-ERROR-003: 大模型配置错误
- **优先级**：中
- **测试步骤**：
  1. 在设置中输入错误的API密钥
  2. 发送任意对话
  3. 查看错误提示
- **预期结果**：
  - 提示："大模型调用失败，请检查API密钥配置"
