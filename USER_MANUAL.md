## 1. 安装指南

### 1.1 Windows系统安装
1. 下载安装包：`Agent助手_Setup.exe`
2. 双击运行安装包
3. 按照提示点击"下一步"
4. 可以选择安装位置（默认即可）
5. 点击"安装"，等待安装完成
6. 勾选"立即运行Agent助手"，点击"完成"

### 1.2 macOS系统安装
1. 下载安装包：`Agent助手.dmg`
2. 双击打开dmg文件
3. 将"Agent助手"图标拖拽到"应用程序"文件夹
4. 打开 Launchpad，找到"Agent助手"点击运行
5. 如果提示"无法打开"，请按以下操作：
   - 打开"系统设置" → "隐私与安全性"
   - 往下翻，找到"安全性"部分
   - 点击"仍要打开"，输入电脑密码确认

## 2. 首次启动配置

第一次打开软件需要简单配置几个参数，只需要配置一次，以后会自动记住。

### 2.1 基础配置
打开软件后，点击右上角的"设置"按钮（齿轮图标）：

1. **云管家工作台地址**
   - 输入你们公司的云系统网址
   - 比如：`https://workspace.你们公司.com`
   - 不知道的话问IT管理员要

2. **大模型API密钥**
   - 这个是AI功能的钥匙，找管理员要
   - 格式类似：`sk-xxxxxxxxxxxxxxxxxxxx`
   - 复制粘贴到输入框里就行

3. **系统认证Token**
   - 登录你们的云系统后，可以在个人设置里找到
   - 或者问IT管理员要
   - 填了这个就不用每次登录系统了

### 2.2 执行模式选择
在主界面左上角可以选择执行模式，推荐使用默认的"智能模式"：
- 🚀 **智能模式**（推荐）：自动选最快的方式执行，优先调用接口，失败了自动用网页操作
- ⚡ **接口优先**：只使用接口方式，速度最快，没有对应接口会问你
- 🌐 **网页操作**：只模拟人工操作网页，兼容性最好

配置完点击"保存设置"就可以开始使用了！

## 3. 基本使用方法

### 3.1 界面介绍
软件界面分为左右两部分：
- **左边：对话窗口**：你在这里发指令，AI在这里回复结果
- **右边：网页窗口**：显示你们公司的云系统，和你平时用的浏览器一样

### 3.2 怎么发指令
1. 在左下角的输入框里用自然语言输入你要做的事
2. 点击"发送"按钮，或者按回车键
3. 等待AI处理，处理过程中会显示加载动画
4. 处理完成后，AI会告诉你结果，网页也会自动完成对应的操作

### 3.3 查看执行详情
AI回复的消息里会显示执行信息：
- 执行模式：用了哪种方式执行（接口/网页）
- 执行时间：花了多少秒
- 执行结果：成功还是失败，返回的数据
- 如果失败了，会告诉你失败原因和解决方法

## 4. 功能说明

### 4.1 AI自动执行任务
你只需要用平时说话的方式说清楚要做什么就行，比如：
- "导出这个月的订单报表"
- "帮我查一下客户张三的联系方式"
- "创建一个新的采购订单，供应商是XX公司，金额5000元"
- "把上周的考勤记录导出来"

AI会自动理解你的需求，选择最合适的方式完成。

### 4.2 双执行模式说明
软件有两种执行方式，会自动选择：
1. **接口调用**：直接和系统后台通信，速度最快，几秒钟就能完成
2. **网页自动化**：模拟人工操作网页，点击、输入、提交，和你自己操作一样

智能模式下会优先用接口，接口不行自动用网页，不用你管。

### 4.3 历史记录
所有对话和执行记录都会保存下来，你可以随时查看：
- 滚动对话窗口就能看历史记录
- 点击"清空对话"按钮可以删除所有记录

## 6. 常见问题解答

### Q1：AI听不懂我的需求怎么办？
A：尽量把需求说清楚，包含必要的信息，比如时间、类型、编号等。比如不说"导出报表"，要说"导出3月份的销售报表"。

### Q2：执行失败了怎么办？
A：AI会告诉你失败原因，常见的解决方法：
- 如果是"网络错误"：检查一下网络是不是正常
- 如果是"身份验证失败"：去设置里重新填一下Token
- 如果是"元素未找到"：切换到"网页操作"模式再试一次
- 如果还是不行，联系管理员处理

### Q3：为什么有时候快有时候慢？
A：用接口执行的时候很快，几秒钟就好；用网页操作的时候会慢一点，因为要模拟人工点击、等待页面加载，和你自己操作的速度差不多。

### Q4：软件会不会泄露我的数据？
A：所有数据都保存在你自己的电脑上，不会上传到任何第三方服务器，放心使用。

### Q5：可以同时执行多个任务吗？
A：建议等一个任务执行完再发下一个，这样不会出错。

### Q6：怎么更新软件？
A：有新版本的时候，软件会自动提示你更新，点击确认就会自动下载安装。

### Q7：macOS提示"无法打开，因为来自身份不明的开发者"怎么办？
A：
1. 右键点击应用图标，选择"打开"
2. 在弹出的提示里再次点击"打开"
3. 以后就可以正常打开了

### Q8：Windows提示"有病毒"怎么办？
A：这是杀毒软件误报，把软件添加到信任列表就行，我们的软件绝对安全。
